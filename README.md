# Employee Management System (EMS)

## 🚀 Tech Stack
- Backend: Spring Boot
- Frontend: React (Vite)
- Database: MySQL
- API Testing: Postman

## ✨ Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee

## 🔗 API Endpoints
- GET /api/employees
- POST /api/employees
- PUT /api/employees/{id}
- DELETE /api/employees/{id}

## ▶️ Run Locally

### Backend
mvn spring-boot:run

### Frontend
npm install
npm run dev
